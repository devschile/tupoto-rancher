# tupo.to Rancher

Rancher template for tupo.to

## Install

In Rancher's UI, go to **Admin/Settings** and add a new custom catalog:

| Name   | URL                                            | Branch |
| ------ | ---------------------------------------------- | ------ |
| tupoto | https://github.com/lgaticaq/tupoto-rancher.git | master |

## Templates

* **tupoto**: All services for production environment
